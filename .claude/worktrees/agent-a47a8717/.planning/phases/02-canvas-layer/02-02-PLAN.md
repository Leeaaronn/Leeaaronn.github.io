**initGlobe() implementation:**

1. **Renderer setup:**
   - `new WebGLRenderer({ alpha: true, antialias: true })`
   - Size: `window.innerWidth` x `window.innerHeight`
   - `renderer.setPixelRatio(Math.min(window.devicePixelRatio, 2))` — RESP-02 cap
   - Canvas CSS: `position: fixed; top: 0; left: 0; z-index: 1; pointer-events: none;`
   - z-index 1 puts globe ABOVE stars (z-index 0) but BELOW page content
   - Append `renderer.domElement` to `document.body`

2. **Camera setup:**
   - PerspectiveCamera(45, window.innerWidth / window.innerHeight, 0.1, 100)
   - Position: `camera.position.set(0, 0, 5)`
   - The module-level `CAMERA_Z_DEFAULT = 5` constant matches this initial position

3. **Scene + lighting:**
   - `new Scene()`
   - `new AmbientLight(0x333333)` — dim ambient so dark side is visible
   - `new DirectionalLight(0xffffff, 1.5)` at position (5, 3, 5)

4. **Earth sphere (GLOB-01):**
   - Create a `THREE.Group()` and assign to `globeGroup`, add to scene
   - `new SphereGeometry(GLOBE_RADIUS, 64, 64)` — 64 segments for smooth sphere
   - Load textures using TextureLoader:
     - Diffuse map: `https://unpkg.com/three-globe/example/img/earth-blue-marble.jpg`
     - Bump map: `https://unpkg.com/three-globe/example/img/earth-topology.png` with bumpScale 0.05
     - Specular map: `https://unpkg.com/three-globe/example/img/earth-water.png`
   - Material: `new MeshPhongMaterial({ map: diffuseTexture, bumpMap: bumpTexture, bumpScale: 0.05, specularMap: waterTexture, specular: new THREE.Color(0x222222), shininess: 25 })`
   - Create mesh, add to globeGroup
   - Assign to `earthMesh`

5. **Atmospheric glow (GLOB-02):**
   - Create a slightly larger sphere: `new SphereGeometry(GLOBE_RADIUS * 1.15, 64, 64)`
   - Use a custom ShaderMaterial for the glow effect:
     ```
     vertexShader:
       varying vec3 vNormal;
       void main() {
         vNormal = normalize(normalMatrix * normal);
         gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0);
       }
     
     fragmentShader:
       varying vec3 vNormal;
       void main() {
         float intensity = pow(0.65 - dot(vNormal, vec3(0.0, 0.0, 1.0)), 4.0);
         gl_FragColor = vec4(0.376, 0.647, 0.980, 1.0) * intensity;
       }
     ```
     Note: RGB (0.376, 0.647, 0.980) is #60a5fa in normalized floats.
   - Material properties: `{ side: BackSide, blending: AdditiveBlending, transparent: true }`
   - Create mesh, add to globeGroup
   - Assign to `atmosphereMesh`

6. **LA marker (GLOB-03):**
   - Convert lat/lon to 3D position on sphere surface:
     ```
     function latLonToVec3(lat, lon, radius) {
       const phi = (90 - lat) * (Math.PI / 180);
       const theta = (lon + 180) * (Math.PI / 180);
       const x = -radius * Math.sin(phi) * Math.cos(theta);
       const y = radius * Math.cos(phi);
       const z = radius * Math.sin(phi) * Math.sin(theta);
       return new THREE.Vector3(x, y, z);
     }
     ```
   - Create marker at `latLonToVec3(34.05, -118.25, GLOBE_RADIUS * 1.01)` — slightly above surface
   - Marker geometry: `new SphereGeometry(0.03, 16, 16)`
   - Marker material: `new MeshPhongMaterial({ color: 0x60a5fa, emissive: 0x60a5fa, emissiveIntensity: 0.8, transparent: true })`
   - Add to globeGroup
   - Also create a pulsing ring around marker:
     - Use a second larger sphere or a RingGeometry at the same position
     - Simpler approach: use the marker mesh opacity pulsing in the animation loop
   - Assign to `markerMesh`

7. **Animation loop (GLOB-04):**
   - `clock = new Clock()`
   - Use `renderer.setAnimationLoop(animate)`
   - In `animate()`:
     - `globeGroup.rotation.y += 0.002` — smooth slow Y rotation
     - Pulse the marker: `markerMesh.material.emissiveIntensity = 0.5 + Math.sin(clock.getElapsedTime() * 3) * 0.5`
     - Also pulse marker scale: `const pulse = 1 + Math.sin(clock.getElapsedTime() * 3) * 0.3; markerMesh.scale.set(pulse, pulse, pulse);`
     - `renderer.render(scene, camera)`

8. **Resize handler:**
   - Listen for `window resize` event
   - Update camera aspect, projection matrix, and renderer size
