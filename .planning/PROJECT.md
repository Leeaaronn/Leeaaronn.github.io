# Portfolio Website — Aaron Lee

## Vision
Personal portfolio site for Aaron Lee, entry-level Data Analyst / Data Operations. A visually striking, space-themed single-page site with a 3D Earth globe, star field background, and scroll-snap navigation. Deployed to GitHub Pages at leeaaronn.github.io.

## Owner
Aaron Lee — B.S. Computer Science, Cal State Fullerton (Jan 2025). Based in Rosemead, CA (Los Angeles area).

## Stack
- **Build:** Vite + vanilla JavaScript (NO React, NO frameworks)
- **3D:** Three.js
- **Styling:** Hand-written CSS with custom properties (NO Tailwind)
- **Deployment:** GitHub Pages from main branch

## Design Principles
- **Color palette:** Black (#000000) + dark greys + white + light blue accent (#60a5fa). HARD RULE: NO green, NO purple, NO orange.
- **Typography:** Instrument Serif (headings), Syne (body), JetBrains Mono (code/labels). Minimum 16px body text. Hero title 60px+.
- **Globe:** Must use real NASA Blue Marble texture — never procedural noise.
- **Stars:** Fixed star field visible on ALL sections, never hidden.
